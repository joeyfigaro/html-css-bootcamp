HTML

Overview

HTML documents are built using tags - tags always begin with a left-angled bracket and end with a right-angled bracket.

If a tag wraps around text or content, it has a closing counterpart, which contains a forward slash.
Example: <p>Hi! I'm a paragraph</p>

Document Tags

  • <html> the very first and very last tag in an html document
  • <head> the second tag in your document - wrapped around document information that is not intended to show up in the body of the page
  • <title> goes inside the <head></head> section of a page and controls the title displayed within the browser
  • <body> all of the page content goes here

Layout Tags

  • <h1> - <h6> used to create page structure - browsers use headings to interpret page layout example
  • <div> generic container, used to wrap content - has no semantic value
  • <p> paragraph tag, used to denote paragraphs of text
  • <a> anchor tag, used to link to pages and resources
  • <img> image tag, used to embed images within a page
  • <ul> unordered list, used to make ...unordered lists
  • <br> line break, used to force content to continue on the next line
  • Tag reference sheet: http://goo.gl/Peo5pr
