Overview

Cascading Style Sheets

A style sheet language used for describing the look and formatting of a document written in a markup language, like HTML.

  • CSS
  • ...can be written within your HTML document or inside a seperate file. Seperate files are called stylesheets, and are preferred.
  • ...is used for font sizes, colors, layout, and much more.
  • ...is mostly written using two HTML attributes: class and id

HTML Attributes

By adding these attributes to our HTML elements, we have an easy way to target them in our stylesheet. We can think of these attributes as identifiers

CSS

Moving Parts

CSS Rules are made of the following:

  • selector
    The target HTML element the rule applies to
  • property
    The change you're making to the target
  • value
  • curly braces and semicolons
    selector {
      property: value;
      property: value;
      property: value;
    }
A CSS rule begins at the selector and ends at the closing curly brace.
