The Web

1

Browsers

  • Browsers are programs that render content on the web, most often HTML pages and related resources.
  • HTML is a markup language that describes a webpage. When you're on a webpage, you can "view source" to see the HTML that the browser is rendering. You can also use a browser's developer tools to view and modify HTML while on the page.
2

Browsers

Various companies and organizations provide browsers, and each browser varies in how it displays webpages.

4

Servers

Servers are just programs running on a computer that serve up content.

Sometimes they serve up static files, and sometimes they're scripted using "server-side" programming languages (PHP, Python, Perl, Java, etc.) to serve more dynamic pages. A static page will forever display the same content - dynamic pages change based on a set of specific parameters.
